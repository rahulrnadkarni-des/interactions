# Design System

Derived from rahulrn.framer.website

## Font
- Family: `Fira Sans` (400, 500, 600, 700)
- Loaded via Google Fonts in `index.css`

## Colors
| Token | Value | Use |
|---|---|---|
| `--bg` | `#fff` | Page background |
| `--surface` | `#f8f8f8` | Card / panel background |
| `--surface-alt` | `#f2f5fa` | Stage / canvas background |
| `--border` | `#d3d8e3` | Standard borders |
| `--border-light` | `#e8eaf0` | Subtle dividers |
| `--text` | `#222` | Primary text |
| `--text-secondary` | `#777` | Secondary / descriptions |
| `--text-muted` | `#ccc` | Timestamps, labels |
| `--accent` | `#ff4603` | Brand orange — CTAs, highlights |
| `--accent-light` | `#ffeee8` | Accent tint background |
| `--accent-hover` | `#e53d00` | Accent on hover |
| `--blue` | `#0041ca` | Links, info |
| `--blue-light` | `#e3f2ff` | Blue tint background |

## Type Scale
| Token | Size |
|---|---|
| `--text-xs` | 12px |
| `--text-sm` | 14px |
| `--text-base` | 16px |
| `--text-lg` | 20px |
| `--text-xl` | 24px |
| `--text-2xl` | 32px |
| `--text-3xl` | 48px |

## Spacing
4px base unit: `--space-1` (4px) through `--space-16` (64px)

## Radius
| Token | Value |
|---|---|
| `--radius-sm` | 6px |
| `--radius-md` | 10px |
| `--radius-lg` | 16px |
| `--radius-full` | 9999px |

## Shadows
- `--shadow-sm` — subtle lift (cards at rest)
- `--shadow-md` — hover state lift
